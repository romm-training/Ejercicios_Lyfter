Métodos de la clase file en Python

Método Descripción
read() Lee todo el contenido del archivo como un string.
readline() Lee una sola línea del archivo.
readlines() Lee todas las líneas y devuelve una lista de strings.
write() Escribe un string en el archivo.
writelines() Escribe una lista de strings en el archivo (no añade saltos de línea).
seek() Mueve el puntero del archivo a una posición específica.
tell() Devuelve la posición actual del puntero en el archivo.
flush() Fuerza a que se escriban en disco los datos que están en el buffer.
close() Cierra el archivo y libera los recursos asociados.
truncate() Reduce o expande el archivo al tamaño especificado.
